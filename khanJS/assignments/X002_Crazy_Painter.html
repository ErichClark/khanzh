Project: Crazy Painter

You just learnt how to draw shapes under your mouse. Now test your creativity by making the craziest painting program you can.

Experiment with using mouseX and mouseY with the shape commands, fill(), and stroke(). Change their sizes and colors. Check out the documentation for each to get ideas of what you can do. There are some secret extra parameters!

If you use your painter to make a beautiful painting, you could screenshot it and email or print the screenshot.

After writing code for your project, please request an evaluation to get points.
These evaluations will be public, so that we can all learn from each other.

Starting code: 

noStroke();
var draw = function() {
    fill(mouseX, 0, 0);
    ellipse(mouseX, mouseY, 10, 10);
};
