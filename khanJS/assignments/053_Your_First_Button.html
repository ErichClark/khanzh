Challenge: Your First Button
挑戰：你的第一個按鈕

Step 1: Is the mouse pressed?
第一步：滑鼠有按下去嗎？

Buttons often change color when they're pressed, so that the user realizes they've pressed something. In this challenge, you'll change the code so that this button turns red when you press it. For this first step, add an if that colors the button red when you press the mouse anywhere on the canvas.
點擊按鈕之後，按鈕通常會改變顏色，讓使用者知道確實有點到東西。在這個挑戰裡，你需要更改程式碼，使按鈕被點擊後轉為紅色。首先，加入if條件，使得滑鼠在畫布任意處點擊後，按鈕即轉為紅色。

Step 2: But WHERE is the mouse pressed?
第二步：滑鼠游標到底點到哪裡呀？

Now you'll make it act more like a button - it should only change color if the user presses on the button, not outside of it. Add another check in the if to make sure that the current y position of the mouse is in the top half of the canvas. Draw it out on a piece of paper if it helps.
