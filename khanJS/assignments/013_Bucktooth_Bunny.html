Challenge: Bucktooth Bunny
挑戰：大板牙兔

Use the power of variables to give our bunny bigger eyes and teeth!
運用變數的力量來給我們的兔子Bucky大一點的眼睛和牙齒！

Step 1: Change the eyes
第一步：改眼睛

Bucky's eyes are drawn as ellipses, and the last two numbers represent their width and height. Create a variable that stores the eye size and use that to change the size of both eyes. Once you've done that, have fun dragging the slider to make HUGE eyes and teeeeny tiny eyes.
Bucky的眼睛是用橢圓形畫成的，且最後兩個數字代表它們的寬度和高度。寫一個變數來儲存眼睛大小然後用它來改變兩隻眼睛的大小。完成之後，盡情地拖曳滑桿來做出超大和超小的眼睛吧。 

var       =      ;
ellipse(170, 150,      ,      ); // left eye
ellipse(230, 150,      ,      ); // right eye

Step 2: Longer teeth!
第二步：長一點的牙齒！

Bucky's buckteeth are drawn as rectangles, and the last number is the height of the rectangle. Create a variable to store the length of the teeth so that you can change the length of both buckteeth at once.
Bucky的大板牙是用矩形畫成的，而最後的數字代表矩形的高度，寫一個變數來儲存牙齒的長度，以便你能夠一次同時改變兩隻大板牙的長度 

var       =      ;
rect(185, 200, 15,      ); // left tooth
rect(200, 200, 15,      ); // right tooth
