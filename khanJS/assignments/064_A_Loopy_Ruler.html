Step 1: Loop it!
第一步：迴圈！

See how we repeat the text() command 7 times, to draw ruler marks for the x coordinates along the top? Let's turn that into a while loop instead! Remember to think about:
記得之前我們怎麼讓文字()命令重複執行7次，以便在上方X座標畫上尺規標記嗎？讓我們把它轉換成一個「條件式」迴圈吧！記得想想：

  1. What do you want to repeat each time?
    每一次的程式執行，你想要重複什麼？
  2. What value do you want to change each time?
    每次改變哪個值？
  3. When do you want to stop looping?
    什麼時候停止迴圈？
