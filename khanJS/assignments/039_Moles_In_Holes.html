Challenge: Moles In Holes

Step 1: Draw a mole!

Our grass is filled with holes, but has no moles. Thankfully, we've already defined a mole drawing function, so all you have to do to get a mole to show up is call that function. Try it! 

drawMole();

Step 2: Parameter-ize the mole

Now, we want to be able to customize where our mole is drawn, so that we can move him around or make more moles in the other holes. Change the function so that it takes two parameters, the x and y position, and pass them when you call the function at the bottom. Change the values so the mole's in the middle of the hole. 

var drawMole = function(_____ , _____) {
	.........
};
drawMole(_____ , _____);

Step 3: MORE moles!

Now that you made that change, it's easy to add more moles. Call the function three more times, and pass values in that make the moles show up in the holes. 

drawMole(_____ , _____);
drawMole(_____ , _____);
drawMole(_____ , _____);
drawMole(_____ , _____);
