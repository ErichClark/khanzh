Challenge: Exploding Sun

Make the sun get bigger and BIGGER!

Step 1: Prepare to animate!

Right now, this program just draws a sun that never moves (boring!). To prepare to animate it, make a draw() function and move all the shape drawing commands inside of it. 

var sunSize = 30;

var draw = function() { 
	// The sun
	fill(255, 204, 0);
	ellipse(200, 298, sunSize, sunSize;
	
	// The land
	fill(76, 168, 67);
	rect(0, 300, 400, 100);
};

Step 2: Explode the sun!

Now, to make the sun get bigger, add to the sunSize variable inside the draw() function. Make it get bigger by 1 pixel each time it's drawn, by adding 1 to the value of sunSize. Remember you can press 'Restart' to restart the animation. 

var sunSize = 30;

var draw = function() { 
	___ = ___ + 1;
