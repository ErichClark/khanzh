Question 1

What will this code output?
這程式碼將會輸出什麼？

var i = 0;
while(i < 3) {
   println("hi"); // "嗨"
    i++;
}

Note: the println() function prints out a line of text with the value that you pass to it - so if you say println("Hi"), it will output: "Hi"
